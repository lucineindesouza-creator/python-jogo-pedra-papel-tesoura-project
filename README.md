# 🪨📄✂️ Jogo Pedra, Papel e Tesoura

Projeto desenvolvido em Python com o objetivo de praticar conceitos fundamentais da linguagem por meio da implementação do clássico jogo **Pedra, Papel e Tesoura** em modo texto.

O programa permite que o usuário dispute partidas contra o computador, utilizando escolhas aleatórias, validação das entradas, controle de placar e repetição das partidas.

# 🎯 Objetivos

* Praticar estruturas condicionais.
* Utilizar laços de repetição.
* Trabalhar com tuplas e constantes.
* Validar entradas do usuário.
* Utilizar a biblioteca `random`.
* Desenvolver lógica de programação.
* Produzir um projeto organizado para portfólio no GitHub.

# 🚀 Funcionalidades

* Partidas entre usuário e computador.
* Escolha aleatória da jogada do computador.
* Validação das opções informadas pelo usuário.
* Controle de vitórias, derrotas e empates.
* Placar atualizado a cada rodada.
* Possibilidade de jogar várias partidas consecutivas.

# 🛠️ Tecnologias utilizadas

* Python 3
* Biblioteca padrão `random`

# ▶️ Como executar

Clone o repositório:

```bash

```

Acesse a pasta do projeto:

```bash
cd pedra-papel-tesoura
```

Execute o programa:

```bash
python main.py
```
